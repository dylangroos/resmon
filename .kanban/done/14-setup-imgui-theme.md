# Setup ImGui Theme

**Priority:** MEDIUM
**Phase:** 4 - GUI Implementation

## Description
Create a minimal/clean dark theme for the UI.

## Style Goals
- Dark background (#1a1a2e or similar)
- Clean accent color (blue or teal)
- No rounded corners (sharp, minimal)
- Readable monospace font
- Subtle borders, low contrast

## Tasks
- [ ] Create src/gui/theme.h/.cpp
- [ ] Set ImGui color scheme
- [ ] Configure window styling (no title bars, minimal padding)
- [ ] Load a good monospace font (JetBrains Mono, Fira Code)
- [ ] Set consistent widget sizing

## Acceptance Criteria
- Looks clean and professional
- Easy to read at a glance
- Not "default ImGui gray"
